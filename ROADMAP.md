# Sophie Roadmap

## Phase 1 — Foundation

Статус: завершено.

- Bootstrap проекта.
- Async architecture.
- CommandBus.
- RuleBasedRouter.
- WorkerFactory.
- BaseWorker.
- EchoWorker.
- BaseProvider.
- OpenRouterProvider.
- LLMManager.
- CLI Interface.
- Telegram Interface.

## Phase 2 — Capabilities

Статус: в работе.

- In-memory conversation memory.
- Интеграция краткосрочной памяти в Worker pipeline.
- Первые специализированные Worker.
- Tools abstraction для внешних сервисов.
- API interface.

## Phase 3 — Intelligence

Статус: запланировано.

- LLM-based Router.
- Planner.
- Tool selection.
- Более сложные сценарии диалога.
- Улучшенное управление контекстом.

## Backlog

- Долговременная память.
- PostgreSQL-backed storage для устойчивых данных.
- Web interface.
- Observability.
- Rate limiting.
- Улучшенная обработка ошибок провайдеров.
- Расширение набора LLM providers.
